Chapter 3

Series vs Parallel

A string of LED fairy lights arrives in a bag. "Half the lights have gone out," the owner says. "The other half are fine." It's a newer set — not the old incandescent kind — but the story is the same. The owner has already tried pressing the fuse caps on the controller. You plug it in, confirm the lit section, and reach for the multimeter. Before you test anything, you pause to think about how this string is actually wired. That's the thing about fairy lights — there are at least three completely different answers, depending on the design.

Learning goals
  • Tell the difference between series and parallel connections from symptoms and wiring.
  • Predict what happens to voltage and current when a branch opens or shorts.
  • Apply that picture to real appliances with multiple functions or loads.

Two ways to connect things

Everything in a circuit is connected in one of two fundamental ways: in series (one after another, sharing the same current) or in parallel (side by side, sharing the same voltage). Most real circuits use both, in combination. Understanding the difference is one of the core skills of fault diagnosis, because series and parallel circuits fail in completely different ways.

Series circuits

In a series circuit, the components are connected end-to-end in a single chain. There's only one path for current to flow, and it flows through each component in turn.

The key rules for series circuits:

  • The same current flows through every component (there's only one path)
  • The voltages across each component add up to the total supply voltage
  • The total resistance is the sum of all individual resistances
  • If any component fails open-circuit, the entire circuit goes dead

These rules are really consequences of there being only one loop. If charge has only one path available, every coulomb that leaves the source must pass through every component in turn, so the current is the same everywhere in the chain. And if the source gives that charge a fixed total energy push, that energy has to be spent somewhere along the loop — shared out as voltage drops across the individual components, whose drops add back up to the supply.

That last point explains the old incandescent fairy lights. All the bulbs were wired in series, with each bulb designed for a fraction of the mains voltage. If you had 40 bulbs on 230V, each one was designed to drop about 5.75V. The problem: one bulb blows, and the entire string goes dark — because the single current path is broken.

Tempting but wrong

"If one bulb in a series string fails, the other bulbs should get more voltage and glow brighter." That would only happen if current could still flow. In a true series string with one bulb open, the path is broken and the current everywhere falls to zero.

From the workbench

The old incandescent Christmas light repair trick — working your way along the string testing each bulb for continuity — was tedious because there were sometimes 50 or more in series. The modern solution was to put a shunt inside each bulb base: a small device with high resistance in normal operation, but which shorts out if the filament blows (converting the open circuit to a near-short circuit, allowing the rest of the string to stay lit). The shunt is why you'd sometimes find a "dead" bulb that made the rest of the string work perfectly — the bulb was dark, but the shunt was doing its job.

Voltage dividers

Series resistances divide voltage in proportion to their values. This is called a voltage divider and it's useful to understand because it appears everywhere — in adjustable heating controls, speed controls, and sensor circuits.

If two resistors are in series across a 230V supply, and one has resistance R₁ and the other R₂, the voltage across R₂ is:

V₂ = 230 × R₂ ÷ (R₁ + R₂)

You don't need to calculate this often at the repair table, but understanding that series resistance drops voltage gives you an important diagnostic insight: if there's unexpected resistance in a series path (a corroded contact, a partial break in a cable), there will be an unexpected voltage drop across it. That voltage drop is measurable, and it's how you find hidden high-resistance faults.

Parallel circuits

In a parallel circuit, the components are connected side by side — each one runs from the live conductor directly to neutral, independently of the others. Each component gets the full supply voltage, and each draws its own current. The currents add up at the junction.

The key rules for parallel circuits:

  • The voltage across every component is the same (all connected between the same two points)
  • The total current is the sum of the currents through each branch
  • The total resistance is lower than any individual resistance (more paths = less resistance)
  • If one component fails open-circuit, the others continue working normally

Again, the "rules" follow from the geometry of the circuit. Each branch is connected to the same two nodes, so each one sees the same potential difference between those nodes — the same voltage. But the charge now has multiple routes available, so the total current from the source is simply the sum of what each branch draws. More branches means more available routes for charge, which is why the overall resistance falls instead of rises.

Every wall socket in your house is in parallel with every other socket. The mains voltage — 230V — appears at all of them simultaneously. Plugging in more appliances adds more parallel branches and draws more current from the supply, but doesn't change the voltage available to any individual appliance.

Predict before measuring

If one branch in a parallel circuit goes open-circuit, what should happen to the voltage across the other healthy branches: stay the same, rise, or fall to zero? Make that prediction before testing fairy lights or multi-function appliances.

What you've just described — voltage staying the same across all parallel branches — is Kirchhoff's Voltage Law in action. That's why you can plug in a kettle, a lamp, and a phone charger at the same time and each gets 230V regardless of the others.

The underlying principle

Parallel resistances share the same voltage but split the current. The formula for combining two parallel resistances is 1/R_total = 1/R₁ + 1/R₂. This is why adding more parallel paths always reduces total resistance — you're giving current more routes to take. In the extreme case (adding a zero-resistance short circuit in parallel with anything), the total resistance drops to zero. This is exactly what a short-circuit fault does: it provides a near-zero-resistance path in parallel with the intended load, bypassing it and drawing enormous current from the supply.

The fairy lights question

Back to the LED fairy lights on the table. There are actually three ways this design might be wired:

All in series: One break kills everything. But with 40–100 LEDs, each LED only needs 2–3V, and the total drops to 230V. Likely if the entire string is dead.

All in parallel with current-limiting resistors: Each LED gets 230V with a resistor to drop the excess. One LED failing open-circuit doesn't affect the others. But this is power-inefficient and not common for mains-connected strings.

Groups in series, groups in parallel: The most common design for modern LED strings. The string is split into sections, each section being a series chain of LEDs. The sections are wired in parallel. One section failing open-circuit (a dead LED) takes out that section only — which explains exactly what you're seeing: half the string dark, half lit.

From the workbench

LED fairy light repair is often more trouble than it's worth, but the diagnostic process is instructive. For a partially-dark string with sections, find the boundary between working and dead sections. Measure voltage at the first dark LED: if you have voltage on one side but not the other, that LED is open-circuit (failed), blocking the series chain. A cheap LED tester or the diode setting on your multimeter can identify the failed LED without desoldering. In practice, if the string is still under warranty or cheaply replaceable, recycling and replacing is often the honest recommendation.

Series and parallel in real appliances

Once you know to look for series and parallel structures, you'll find them everywhere.

Heating elements in series

Some appliances use two heating elements that can be switched between series and parallel to change power output. A simple two-power-level toaster or grill might wire both elements in series for a low-heat setting (each element gets ~115V, total power roughly a quarter of maximum) and in parallel for full power (each element gets 230V). The switch literally reconfigures the circuit topology. If you're testing such an appliance and it seems to heat less than expected on one setting, check whether the series/parallel switching mechanism is working correctly.

Motors with series resistance

Older appliances sometimes used a series resistor to reduce motor speed — lower voltage across the motor means lower speed. A food mixer with a "slow" setting might have a resistor inserted in series with the motor on that setting. The resistor wastes energy as heat (it's an inefficient method of speed control, but cheap to implement). If the slow setting suddenly runs at full speed, suspect an open-circuit resistor — the motor is now getting full voltage with nothing in series.

Protection devices in series

Every fuse, thermal cutoff, and switch in an appliance is wired in series with the load it protects. This is essential: the protecting device must carry the same current as the load — which means it sees the fault current when things go wrong. A fuse in parallel with the load would never blow, because it wouldn't carry the fault current. Always in series.

Combining series and parallel: the practical picture

Most real circuits are combinations. Consider a simple electric iron: the supply enters through a fuse (series), then through a thermostat (series), then branches: one branch to the main heating element, another branch to a small pilot light (parallel). Each branch draws its own current. The fuse and thermostat protect the whole circuit — they're in series with both branches combined.

When you trace a fault in a combined circuit, you first identify which part of the circuit the symptom points to. If the pilot light is on but the element isn't heating: the parallel branch to the element is open somewhere (the element, or a connection to it). If neither light nor heat: the fault is in the series section before the branch — the fuse, the thermostat, the power cable, the switch.

From the workbench

A practical trick for sorting out which topology you're looking at: unplug the appliance, set your multimeter to continuity or low resistance, and measure from terminal to terminal on each component. Components in series will show a path through each other in sequence. Components in parallel will show a low-resistance path from either terminal to either terminal (because both ends are connected to the same two points). With the circuit unpowered, you can safely trace the topology before you ever plug it in.

Short circuits as parallel paths

A short circuit is, technically, a very low-resistance path that appears in parallel with the intended load. When insulation fails and a live conductor touches neutral, there's suddenly a near-zero-resistance path from 230V to 0V, in parallel with everything else in the circuit. The current that flows through this path is enormous (V ÷ R, where R is very small) — far more than the fuse or breaker is rated for. The protection device trips.

This is why a short circuit is different from an open circuit: open circuits produce silence, nothing works, no current flows. Short circuits produce noise, sparks, blown fuses, and tripped breakers — because enormous current is briefly flowing through the wrong path.

The underlying principle

The key insight is that in a parallel circuit, adding more paths reduces total resistance. A short circuit is the extreme case: a zero-resistance path dominates the parallel combination completely, bringing total resistance to near zero regardless of what else is in parallel with it. This is Ohm's law in action: I = V/R, and as R approaches zero, I approaches infinity. In practice, real circuits have some resistance in their wiring and source impedance that limits the current, but it can still reach hundreds or thousands of amperes in the milliseconds before the protection trips.

The LED fairy lights end up being a group-series design — one section of six LEDs has a failed unit. You note the section and hand the string back with an honest assessment of repairability. The owner decides it's not worth it, and the string goes in the recycling box. That's a fair outcome: you've understood the fault and can explain it. The fan with the burning smell has been sitting at the next table for the last twenty minutes, and it's time to take a look at what's heating up inside it.

Remember this
  • Series means one current path: one break can stop everything.
  • Parallel means shared supply: one failed branch can leave the others working.
  • A short circuit is often an unintended parallel path with very low resistance.
  • If only one function fails, start by asking which parts that function does not share with the others.

Check your understanding

Q1 You open a travel kettle and find two identical heating elements. On the low-power setting they're in series; on high power, in parallel. If each element has a resistance of 80 ohms, what is the total resistance and current draw at 230V on each setting? Show answer

Series: Total resistance = 80 + 80 = 160 ohms. Current = 230 ÷ 160 = 1.44A. Power = 230 × 1.44 ≈ 330W.

Parallel: Total resistance = 1 ÷ (1/80 + 1/80) = 40 ohms. Current = 230 ÷ 40 = 5.75A. Power = 230 × 5.75 ≈ 1322W — roughly four times higher. This illustrates why parallel wiring is used for full power: the same elements, rewired in parallel, produce four times the power output.

Q2 A food mixer has a low-speed setting that uses a series resistor to reduce motor voltage. The resistor is rated 30 ohms. On the low setting, the motor runs at full speed and the series resistor is warm. What's the most likely fault? Show answer

The series resistor is still in circuit and still getting warm — so it hasn't failed open. If it had failed open, no current would flow and the motor would stop entirely on that setting. The motor running at full speed despite the resistor being present suggests the speed-selector switch is bypassing the resistor, connecting the motor directly to full supply voltage. The switch contact that's supposed to route current through the resistor on the low setting is probably shorted or welded closed. Check the selector switch continuity — it's not properly inserting the resistance into the circuit.

Q3 An electric iron has both a heating element and a pilot light. The pilot light works normally, but the element doesn't heat. Where in the circuit is the fault most likely to be? Show answer

The pilot light and heating element are in parallel with each other — they share the same supply points. The pilot light working confirms that: (a) the supply is getting to the junction point where they branch, and (b) the common series elements (fuse, power cable, switch) are all intact. The fault must therefore be in the branch containing the heating element only — most likely the element itself (open circuit), a thermostat in series with the element but not the pilot light, or a break in the wiring to the element. The pilot light is acting as a useful diagnostic indicator: it's telling you the problem is downstream of the branch point.

Q4 Why is it essential that fuses and thermal cutoffs are wired in series with the load they protect, rather than in parallel? Show answer

A protection device works by interrupting current flow when that current exceeds a safe level. For the device to sense the current, it must carry that current — which means it must be in series with the load. In a series connection, the same current flows through both the protection device and the load. If the load draws too much current (fault condition), the fuse or thermal cutoff also carries that current and trips or blows, cutting power to the load. In parallel, the protection device would see only the small fraction of current that flows through it — essentially none, since the low-resistance load path takes almost all the current. It would never operate on a fault, providing no protection at all.

Q5 A two-slot toaster wires both elements in each slot in series on the "gentle" setting and in parallel on "normal." One element in the right slot has failed open-circuit. What happens on each setting for that slot? Show answer

On the gentle setting (series): both elements are in a single chain. One element being open-circuit breaks the chain entirely — no current flows through either element. The right slot is completely dead on the gentle setting.

On the normal setting (parallel): each element has its own independent path to the supply. The failed element's branch is open, so that element doesn't heat — but the working element's branch is intact and unaffected. The right slot still heats on normal, but at roughly half its rated power. This is why the fault shows up as "the bread comes out pale" rather than "nothing happens" — a partial failure that's easy to miss without understanding series vs parallel topology.
